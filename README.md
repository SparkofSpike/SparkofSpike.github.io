# Spike · 星邃之径

> 深空之中，微光不灭。

Spike 的个人主页。暗黑星空 / 克苏鲁风格，展示一个 FPS 魔怔人、Galgame 绝症患者、洛氏恐怖信徒、野班子程序员的数字栖身之所。

## 技术栈

- **Vue 3** + **TypeScript**
- **Element Plus**
- **Vite** 构建工具
- Canvas 动态星空背景（星座连线 + 鼠标交互 + 五彩星星）
- GitHub Actions 自动部署到 Pages

## 本地开发

```bash
npm install
npm run dev      # 开发模式 http://localhost:5173
npm run build    # 生产构建 → dist/
```

## 部署

推送到 `main` 分支后，GitHub Actions 自动构建并部署到 GitHub Pages。
